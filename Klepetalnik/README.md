# Chat - opis projekta
Projekt je nastal v okviru predmeta programiranje 2. Program je klepetalnik, 
ki se pogovarja s strežnikom na naslovu http://chitchat.andrej.com/. 

# Navodila za uporabo

Ob zagonu programa se odpre pogovorno okno, kamor uporabnik vpiše ime 
s katerim se želi prijaviti. Èe je uporabniško ime že zasedeno, ga program
na to opozori. 

Ko nas program prijavi, se odpre pogovorno okno. Na levi strani med prijavljenimi uporabniki
izberemo tistega, ki mu želimo poslati sporoèilo (èe izberemo možnost 'vsi',
bo sporoèilo poslano vsem prijavljenim uporabnikom). Sporoèilo vpišemo v vrstico
desno spodaj. Sporoèilo pošljemo s pritiskom na tipko enter. 

Uporabnik se odjavi tako, da zapre okno.
